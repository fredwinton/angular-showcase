# angular-showcase
This repository contains an Angular showcase app that showcases my knowledge of Angular. Built using Angular 4, Bootstrap and REST

 ##  Installation:
   Download or clone the angular-showcase folder. Note that since this is Angular, it will depend on Node.js and npm installed.

* Enjoy!
